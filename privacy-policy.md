Privacy Policy

Last updated: 12/1/2025

This Privacy Policy explains how we collect, use, and protect information when users interact with our Pinterest-connected application (“the App”).

1. Information We Collect

Our App may request and access certain Pinterest account information, such as:

Public profile information

Boards created by the user

Pins created or saved by the user

We do not collect personal information outside of what Pinterest provides through its API.

2. How We Use Information

We use Pinterest API data exclusively to:

Allow the user to create and publish Pins

Manage or view their own boards

Improve the user experience within the App

We do not sell, share, or distribute user data to third parties.

3. Data Storage and Security

All API tokens and user data are stored securely.
We do not store any information longer than necessary for the App’s functionality.

4. Third-Party Services

Our App interacts only with Pinterest API.
Users are subject to Pinterest’s Terms of Service and Privacy Policy.

5. User Rights

Users may revoke the App’s permissions at any time through their Pinterest account settings.
Upon revocation, the App stops receiving data immediately.
