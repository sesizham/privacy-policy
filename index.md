# Privacy Policy for Global Heritage Art

**Last updated: December 2025**

This Privacy Policy is provided by **Global Heritage Art** (“Company”, “we”, “us”) for our Pinterest-connected application (“the App”).

## 1. Information We Collect

Our App may request and access certain Pinterest account information, such as:
- Public profile data  
- Boards created by the user  
- Pins created or saved by the user  

We do **not** collect personal data outside of what Pinterest provides through its API.

## 2. How We Use the Data

We use Pinterest API data only to:
- Allow the user to create Pins through the App  
- Manage or view the user’s own boards  
- Improve the user experience  

We do **not** sell, share, or transfer any data to third parties.

## 3. Data Storage & Security

All API tokens and Pinterest data are stored securely.  
We do not store information longer than necessary for the App to function.

## 4. Third-Party Services

Our App communicates only with **Pinterest API**.  
Users are subject to Pinterest’s Terms of Service and Privacy Policy.

## 5. User Rights

Users may revoke App permissions at any time in their Pinterest account settings.  
After revocation, the App immediately stops receiving data.
