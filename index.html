Privacy Policy for Global Heritage Art

Last updated: December 2025

This Privacy Policy is provided by Global Heritage Art (“Company”, “we”, “us”) for our Pinterest-connected application (“the App”).

1. Information We Collect

Our App may request and access certain Pinterest account information, such as:

  • Public profile data
  • User boards
  • User-created or saved Pins

2. How We Use the Data

We use Pinterest API data only to:

  • Create and publish Pins for the user
  • Manage the user’s own boards
  • Improve the App functionality

We do not sell, share, or disclose any data to third parties.

3. Data Storage & Security

All API tokens and data are stored securely and are not retained longer than needed for normal operation.

4. Third-Party Services

The App interacts exclusively with Pinterest API. Users are subject to Pinterest's Terms of Service and Privacy Policy.

5. User Rights

Users may revoke permissions at any time via Pinterest settings. Once revoked, the App stops receiving all data immediately.
